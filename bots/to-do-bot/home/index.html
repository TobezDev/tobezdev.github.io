To-Do Bot

Welcome to To-Do Bot!

To-Do Bot is a Discord bot that helps you manage your tasks seamlessly within your Discord server. Stay organized, create tasks, and collaborate with your server members effortlessly.

What is each bot permission used for?

To-Do Bot requires certain permissions to provide its features. Here's why each permission is necessary:

  • Identify: The bot needs to know who is using the bot and when. The collected details will usually not be stored for very long, however they may be shared with Discord under certain circumstances.
  • Send Messages: This one is self-explanitory. How can you use the bot when it can't respond to you?
  • Embed Links: Same reasons as above, but applies to commands that are used in thread channels.
  • Attach Files: Allows To-Do Bot to view and gather input from users in text channels.
  • Read Message History: Allows To-Do Bot to attach files to messages. This is useful when our team deliver you your privacy report if you request it.
  • Use External Emojis: Some external emojis are used to spice up select messages.
  • Use External Stickers: Same as above but instead applies to stickers.
  • Add Reactions: Sometimes, during maintenance, To-Do Bot will queue commands to execute after the maintenance phase. It will respond to a few messages to acknowledge that command has been queued.
  • Use Slash Commands: To-Do Bot uses cool slash commands to operate instead of the louzy text commands. Cool, right?
